# الهوية البصرية والألوان (Visual Identity Colors) — منصة معين

يوفر هذا المستند مرجعاً شاملاً للألوان المعتمدة للهوية البصرية الخاصة بـ **منصة معين الرقمية**، بما يضمن اتساق التصاميم عبر كافة المكونات البرمجية والواجهات الرسومية في وضعي الإضاءة (Light Mode) والظلام (Dark Mode).

---

## 🎨 لوحة الألوان الأساسية (Core Palette)

تتألف الهوية البصرية لمنصة معين من أربعة ألوان رئيسية تعبر عن طابع المنصة الخدمي والتنموي:

### 1. الأخضر الزمردي الداكن (Primary - Deep Emerald)
* **القيمة (HEX):** `#0A5C4A`
* **المعنى:** يمثل الاستدامة، النمو، والتمكين، وهو اللون الطاغي على الترويسات، الأزرار الأساسية، والروابط الهامة.
* **الاستخدام:** 
  * العناصر الرئيسية النشطة (Primary CTA buttons)
  * الترويسة العلوية في وضع الضوء (Header background)
  * الأيقونات الرئيسية والإشارات التوضيحية.

### 2. الأصفر الذهبي (Secondary - Golden Amber)
* **القيمة (HEX):** `#F9A826`
* **المعنى:** يرمز للثقة، الحيوية، والتميز، ويستخدم لإضفاء روح إيجابية وجذب انتباه المستخدم للعناصر التفاعلية الهامة.
* **الاستخدام:**
  * شارات التنبيهات المميزة (Badges)
  * عناصر التفاعل الفوري وأزرار الاتصال الفرعية.
  * مؤشرات التمييز وأجزاء من الرسوم البيانية.

### 3. الأخضر الفيروزي (Tertiary - Mint Teal)
* **القيمة (HEX):** `#2FAB99`
* **المعنى:** يعكس الشفافية، الحداثة، والوضوح، ويستخدم لتسهيل القراءة ولتصميم المكونات المساندة.
* **الاستخدام:**
  * درجات التدرج اللوني (Gradients)
  * خطوط الرسوم البيانية والحدود (Borders) للمكونات النشطة.
  * الرسائل الترحيبية وتصنيفات البيانات المقروءة.

### 4. الرمادي الفحمي (Brand Neutral - Charcoal Grey)
* **القيمة (HEX):** `#464646`
* **المعنى:** يوفر توازناً بصرياً مريحاً للعين بدلاً من اللون الأسود الصريح، مما يضفي مظهراً عصرياً فاخراً.
* **الاستخدام:**
  * نصوص العناوين الفرعية.
  * الأيقونات الثانوية والحدود الافتراضية.

## 🌓 واجهة الإضاءة والظلام (Theme Modes)

تعتمد المنصة على نظام متغيرات CSS متجاوب مع وضع النظام المختار:

### ☀️ وضع الإضاءة (Light Mode)
* **خلفية التطبيق (`--background`):** `#f8fafc` (رمادي ناصع مائل للزرقة لراحة العين)
* **لون النصوص الرئيسي (`--foreground`):** `#0f172a` (أزرق ليلي داكن جداً لتحقيق أعلى درجات التباين والقرائية وفق معايير WCAG)

### 🌙 وضع الظلام (Dark Mode)
* **خلفية التطبيق (`--background`):** `#0b0f19` (أزرق داكن عميق لتقليل إجهاد العين ليلاً ومنع التوهج)
* **لون النصوص الرئيسي (`--foreground`):** `#e2e8f0` (أبيض حليبي مريح)

## 📊 الألوان في لوحة البيانات والرسوم البيانية (Dashboard & Charts)

تم اختيار ألوان الرسوم البيانية SVG لتنسجم مع الهوية مع الاحتفاظ بدلالات الألوان الوظيفية (Semantic Colors):

| الوظيفة البصرية | قيمة اللون (HEX) | الاستخدام الفني |
| :--- | :--- | :--- |
| **المستحق (Eligible / Success)** | `#10B981` | يمثل التوزيع الإيجابي والنمو للمستفيدين المستحقين. |
| **قيد الدراسة (Pending / Warning)** | `#F59E0B` | يمثل التنبيهات وحالات الدراسة المعلقة. |
| **غير مستحق (Ineligible / Error)** | `#EF4444` | يمثل الاستثناءات، الإلغاء، أو الحالات المرفوضة. |
| **المسوقون ونقاط التوزيع** | `#3B82F6` | يمثل بيانات وإحصائيات المسوقين. |

## 💻 دمج الألوان برمجياً (Implementation)

تم تعريف الألوان مباشرة داخل ملف الإعدادات الخاص بـ TailwindCSS في ملف [globals.css](file:///d:/Ghazaara%20Net/مؤسسة%20بيت%20التمكين%20للتسويق%20-%20المملكة%20العربية%20السعودية/ملفات%20المشروع%20الأصلية%20-%20اعداد%20عبدالداني/moeen-platform/moeen-platform/src/app/globals.css) كالتالي:

```css
@theme {
  --color-primary: #0A5C4A;
  --color-secondary: #F9A826;
  --color-tertiary: #2FAB99;
  --color-brand-neutral: #464646;
}
```

ويمكن استخدامها في المكونات بسهولة عبر الفئات التالية:
- خلفيات: `bg-primary`, `bg-secondary`, `bg-tertiary`
- نصوص: `text-primary`, `text-brand-neutral`
- حدود: `border-tertiary`
- وضع الظلام: `dark:bg-[#0b0f19]`, `dark:text-slate-200`
